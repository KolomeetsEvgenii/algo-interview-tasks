# algo-interview-tasks

Репозиторий с алгоритмическими задачами для подготовки к собеседованиям.

## Цель
Собрать в одном месте популярные задачи по алгоритмам и структурам данных, которые часто спрашивают на технических интервью.

## Формат
- одна задача = один файл
- решения на Python
- задачи разложены по темам
- в каждом файле:
  - условие
  - примеры
  - идея решения
  - сложность `Time / Space`
  - код решения
